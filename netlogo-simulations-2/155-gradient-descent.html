Gradient Descent

Gradient descent is an optimization algorithm that finds minima by following the negative gradient.

Key concepts:

  • Learning rate: Step size toward the minimum
  • Momentum: Accumulates velocity for faster convergence
  • Local minima: Can get stuck in suboptimal solutions
  • Saddle points: Plateaus where gradient is zero
