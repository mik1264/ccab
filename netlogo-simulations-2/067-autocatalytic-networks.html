RAF Theory (Kauffman)

An autocatalytic set is a collection of molecules that mutually catalyze each other's formation from a food set.

  • Reflexively Autocatalytic: Every reaction catalyzed
  • Food-generated: All from food set
  • Catalytic Closure: Self-sustaining network
