Chemical Reaction Rules
2H + O → H₂O (water formation)
C + 2O → CO₂ (carbon dioxide)
C + 4H → CH₄ (methane formation)
H₂O → 2H + O (water splitting, rare)
CO₂ → C + 2O (decomposition, rare)
CH₄ → C + 4H (methane breaking, rare)
