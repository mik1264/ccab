TSL Shaders (Three.js Shading Language)

46 high-quality visualizations showcasing Three.js Shading Language (TSL) - a node-based approach to shader programming. Features foundation experiments with vertex manipulation and noise effects, interactive parametric forms with real-time controls (Lissajous curves with sliders), binary star gravitational simulations, multi-body systems, comprehensive orbital mechanics (Kepler's laws, Hohmann transfers, geosynchronous orbits), exotic phenomena (black holes, gravitational lensing, relativistic effects), and orbital resonances. TSL provides type-safe, maintainable shader code that compiles to both GLSL (WebGL) and WGSL (WebGPU), offering better debugging and cross-platform compatibility than traditional string-based shaders.
