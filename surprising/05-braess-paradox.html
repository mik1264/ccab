Braess's Paradox

Adding a new road can make traffic WORSE for everyone. Sometimes less is more!

The Counterintuitive Truth

In 1968, German mathematician Dietrich Braess discovered that adding road capacity to a network can sometimes increase travel time for all drivers. This has since been verified in real cities - sometimes closing roads actually improves traffic flow!

How Does This Work?

Consider a network with 4,000 drivers traveling from Start to End. There are two routes:

  • Route 1 (Top): Start → A → End
  • Route 2 (Bottom): Start → B → End

The roads have different properties:

Start→A: Time = (drivers/100) minutes (congestion-dependent)
A→End: Time = 45 minutes (fixed)
Start→B: Time = 45 minutes (fixed)
B→End: Time = (drivers/100) minutes (congestion-dependent)

Without the shortcut: Drivers split evenly between routes. Each route has 2,000 drivers, so congestion-dependent roads take 20 min. Total: 20 + 45 = 65 minutes for everyone.

With the shortcut (A→B, free): Everyone now takes Start→A→B→End because it seems faster. But with 4,000 drivers on both congestion roads: 40 + 0 + 40 = 80 minutes!

The Nash Equilibrium Trap

The paradox occurs because of selfish routing. Each driver chooses the route that minimizes their own travel time, without considering the effect on others.

When the shortcut exists:

  • If you're at A, the shortcut (free) + B→End looks better than A→End (45 min)
  • Similarly, everyone takes the congested roads because individually they seem faster
  • Result: Everyone ends up worse off than if the shortcut didn't exist!

This is a Nash Equilibrium - no individual driver can improve their time by changing routes, yet everyone would be better off if they could coordinate.

Real-World Examples

Seoul, South Korea (2003): Demolishing the Cheonggyecheon highway reduced traffic congestion in the area.

New York City: When 42nd Street was closed for Earth Day in 1990, traffic actually improved.

Stuttgart, Germany: A new road made traffic worse until it was closed again.

The lesson: More roads don't always mean less traffic. Sometimes the best solution is removing capacity!
