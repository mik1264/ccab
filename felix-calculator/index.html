How to Use the Felix Calculator

Setting Numbers

Drag the levers up/down or click on the track to set digits 0-9. Each lever controls one decimal place.

Addition

Set a number, then click + (crank forward). The number is added to the result register.

Subtraction

Set a number, then click ‚àí (crank backward). The number is subtracted from the result register.

Multiplication

Set the multiplicand, crank N times for each digit of the multiplier, shifting carriage left between digits.

Division

Set the dividend in result register, set the divisor in input, then crank backward repeatedly.

Carriage

Shift left for higher decimal places (√ó10). The counter shows position-weighted crank counts.
