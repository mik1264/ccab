How to Play

  • Fill each cell with a number from 1 to N.
  • Each row and column must contain each number exactly once (Latin square).
  • Cells are grouped into cages (bold outlines).
  • Each cage shows a target number and an operation (+, −, ×, ÷).
  • The numbers in each cage must combine using the operation to produce the target.
  • Click a cell, then click a number (or use keyboard 1-6).
