Fill cells so connected regions of the same number have that size

How to Play

Fill every empty cell with a number. All cells containing the same number that are connected orthogonally (up/down/left/right) must form a region whose size equals that number.

For example, a group of three connected 3s is valid. A single isolated 1 is valid. Two connected 2s is valid.

Click a cell to select it, then click a number below (or press a key) to fill it. Pre-filled cells cannot be changed.
