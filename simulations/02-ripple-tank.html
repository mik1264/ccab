Interactive Ripple Tank

Click or drag to create ripples. Add barriers to see reflection, refraction, and diffraction. A realistic 2D wave equation simulation demonstrating classical wave phenomena.

Wave Phenomena in Ripple Tank

Wave Equation: ∂²u/∂t² = c² (∂²u/∂x² + ∂²u/∂y²)

This simulation demonstrates fundamental wave behaviors:

  • Reflection: Waves bounce off barriers at equal angles
  • Diffraction: Waves bend around obstacles and through slits
  • Interference: Overlapping waves create patterns of reinforcement and cancellation
  • Refraction: Wave speed changes cause direction changes

Numerical Method: Finite difference with leapfrog integration. Uses CFL condition for stability: Δt < Δx / c
