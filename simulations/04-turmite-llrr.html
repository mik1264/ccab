About LLRR Turmite

  • 4-State System: This turmite uses four distinct states, creating richer patterns
  • LLRR Rule: The symmetric turning pattern:
    • State 0: Turn Left, change to state 1
    • State 1: Turn Left, change to state 2
    • State 2: Turn Right, change to state 3
    • State 3: Turn Right, change to state 0
  • Symmetry: The balanced left-right pattern creates symmetric structures
  • Visual Beauty: Often produces mandala-like circular or spiral patterns
  • Evolution: Watch how the pattern grows from chaos into organized structure
  • Color Gradient: The four states are represented by a smooth color gradient
