SHA-256 Properties

  • Deterministic: Same input always produces same output
  • One-way: Cannot reverse hash to find input
  • Collision resistant: Infeasible to find two inputs with same hash
  • Avalanche effect: Small input change completely changes output
  • Fixed output: Always 256 bits regardless of input size

SHA-256 is used in Bitcoin mining, SSL certificates, password storage, and digital signatures. Finding a collision would require approximately 2^128 operations - more than the number of atoms in the observable universe.
