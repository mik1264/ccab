Mutual Information

Shared information between variables

I(X;Y) = H(X) + H(Y) - H(X,Y)
H(X) 0 bits
H(Y) 0 bits
H(X,Y) 0 bits
I(X;Y) 0 bits
Normalized MI 0
Mutual Information: How much knowing X tells about Y.
Independent: I(X;Y) = 0 (no shared info).
Deterministic: I(X;Y) = H(X) = H(Y).
