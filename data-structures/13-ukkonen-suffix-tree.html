About Ukkonen's Algorithm

Ukkonen's algorithm builds a suffix tree in O(n) time by processing characters online (one at a time).

Key concepts:

• Active Point: Tracks where to insert next suffix
• Suffix Links: Enable O(1) jumps between nodes
• Implicit End: All leaves end at current position
• Remainder: Suffixes still to be inserted
