Core Data Structures (5 visualizations)

Essential computer science data structures with animated operations and complexity analysis.

Binary Search Tree
Interactive BST with insert, delete, and search operations. Visualize tree structure with animated node insertions and three traversal methods (in-order, pre-order, post-order).
Heap Visualization
Min/max heap operations with dual array and tree representations. Watch heapify operations and bubble-up/bubble-down animations in real-time.
Hash Table
Hash table with collision resolution strategies: chaining, linear probing, and quadratic probing. Visualize hash function behavior and bucket distribution.
Graph Traversal (BFS & DFS)
Interactive graph builder with breadth-first and depth-first search algorithms. Create custom graphs or generate random ones to see traversal order visualization.
Trie (Prefix Tree)
Prefix tree visualization for string operations. Insert, search, delete words, and find all words with a given prefix. Perfect for autocomplete demonstrations.

Educational Information

Each visualization includes detailed complexity analysis and real-world applications:

Binary Search Tree
Time: O(log n) average for balanced trees
Use cases: Database indexing, file systems, expression parsing
Heap
Time: O(log n) insert/delete, O(1) peek
Use cases: Priority queues, heap sort, Dijkstra's algorithm
Hash Table
Time: O(1) average case for all operations
Use cases: Caching, database indexing, symbol tables
Graph Traversal
Time: O(V + E) for both BFS and DFS
Use cases: Shortest path, cycle detection, network analysis
Trie
Time: O(m) where m is word length
Use cases: Autocomplete, spell checking, IP routing
