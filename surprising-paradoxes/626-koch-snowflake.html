Von Koch's Monster (1904)

Helge von Koch created this curve to show a function continuous everywhere but differentiable nowhere. Start with a triangle, add bumps to each side, repeat forever. The "snowflake" name came from Kasner's 1940 book!

The Construction Rule

Take each line segment, divide it into thirds, build an equilateral triangle on the middle third, remove the base. Each step multiplies perimeter by 4/3. Do this forever and the perimeter becomes infinite!

Infinite × Zero = Finite?

As iterations increase, we add infinitely many triangles, but each is infinitely small. The sum of all added areas converges: Area = (8/5) × original triangle = 0.6928. Infinite perimeter, finite area!

Cantor's Opposite

The Cantor Set has zero length but uncountably many points. The Koch Snowflake has infinite perimeter but finite area. Both are fractals. Both challenged 19th-century intuitions about dimension!
