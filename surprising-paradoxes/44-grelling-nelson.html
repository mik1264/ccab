Two Types of Words

In 1908, German philosophers Kurt Grelling and Leonard Nelson noticed something peculiar about adjectives. Some adjectives describe themselves‚Äîand some don't.

Autological

A word that describes itself

From Greek: autos (self) + logos (word)

Heterological

A word that does NOT describe itself

From Greek: heteros (other) + logos (word)

Connection to Russell's Paradox

The Grelling-Nelson paradox is closely related to Russell's Paradox about the set of all sets that don't contain themselves.

Think of each word as a set containing all things that satisfy that word's meaning:

  • Autological word = A set that contains itself as an element
  • Heterological word = A set that does NOT contain itself

The question "Is 'heterological' heterological?" becomes identical to Russell's question: "Does the set of all sets that don't contain themselves contain itself?"

What About "Autological"?

Interestingly, while "heterological" creates a paradox, "autological" is merely underdetermined:

  • If "autological" is autological: It describes itself, meaning it IS autological. ‚úÖ Consistent!
  • If "autological" is heterological: It doesn't describe itself, meaning it's NOT autological. ‚úÖ Also consistent!

So "autological" can be either‚Äîit's just that we can't determine which. The word "heterological," however, can be neither.

Why Does This Matter?

The Grelling-Nelson paradox teaches us that:

  • Self-reference is dangerous: When words or concepts refer to themselves, paradoxes can emerge
  • Language has limits: Not all seemingly meaningful questions have answers
  • Definitions can be tricky: Even precise definitions can lead to contradictions
  • Logic needs constraints: We must be careful about what questions we allow in formal systems

The next time you describe a word, ask yourself: does the word describe itself? And if it does... is it really describing itself? ü§î
