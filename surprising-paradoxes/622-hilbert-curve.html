Peano's Shocking Discovery (1890)

Giuseppe Peano proved a 1D line can fill a 2D square—visiting EVERY point! Hilbert (1891) simplified this into an elegant recursive construction. A curve with Hausdorff dimension 2—it's simultaneously a line AND a plane!

Locality Preserving

Unlike the Z-order curve which has large "jumps," the Hilbert curve keeps nearby 1D positions close in 2D. This property makes it ideal for databases—range queries touch fewer disk blocks!

Recursive Beauty

Start with a U-shape. Divide into 4 quadrants, place a rotated copy in each, connect them. Repeat forever. Each iteration doubles the resolution, quadruples the points. At infinity: a continuous curve through every real point!

Modern Applications

Databricks uses Hilbert curves for "liquid clustering." N-body simulations use them for load balancing. Image processing uses them for dithering. All because of that magical locality preservation!
