Can a 1-dimensional line fill a 2-dimensional square? Yes! The Hilbert curve is a continuous path that visits every point in a square. In the limit, it has dimension 2, not 1!

üéØ Locality Preservation

Points close on the curve stay close in 2D! This makes Hilbert curves ideal for database indexing, image compression, and spatial queries. IP address visualizations use this property.

üíª Modern Applications

Google uses Hilbert curves in their S2 geometry library for mapping. Graphics cards use them for texture mapping. They're also used in dithering algorithms and machine learning spatial features!
