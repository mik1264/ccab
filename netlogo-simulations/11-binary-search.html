üîç Binary Search

Binary search efficiently finds a target in sorted data by repeatedly halving the search space. It's one of the most fundamental algorithms in computer science.

How It Works

  1. Compare target with middle element
  2. If equal, found!
  3. If target < middle, search left half
  4. If target > middle, search right half
  5. Repeat until found or range empty

Complexity

  • Time: O(log n) - halves search space each step
  • Space: O(1) for iterative, O(log n) for recursive
  • Requirement: Data must be sorted

Comparison to Linear Search

  • Linear: Check every element ‚Üí O(n)
  • Binary: Halve repeatedly ‚Üí O(log n)
  • 1 million elements: 20 steps vs 1 million

What to Observe

  • How quickly the search space shrinks
  • Maximum steps = log‚ÇÇ(n)
  • Works for any sorted data structure
