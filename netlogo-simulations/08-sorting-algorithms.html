üî¢ Sorting Algorithms

Sorting algorithms arrange data in order. Different algorithms trade off between simplicity, speed, and memory usage.

Common Algorithms

  • Bubble Sort: O(n¬≤) - Repeatedly swaps adjacent elements; simple but slow
  • Quicksort: O(n log n) avg - Divides via pivot; fast but unpredictable worst case
  • Merge Sort: O(n log n) - Guaranteed performance; requires extra memory

Key Concepts

  • Time complexity: How runtime grows with data size
  • Space complexity: How much extra memory is needed
  • Stability: Whether equal elements maintain original order

When to Use Each

  • Bubble: Teaching/learning; tiny datasets
  • Quicksort: General purpose; large arrays
  • Merge Sort: Linked lists; guaranteed performance needed

What to Observe

  • Count comparisons and swaps for each algorithm
  • Notice speed differences on larger datasets
  • Try sorted, random, and reversed input
