Schelling Segregation Visualizations

Understanding Emergence and Tipping Points

Thomas Schelling's segregation model (1969) demonstrates one of complexity science's most profound insights: mild individual preferences can create strong collective patterns through emergent dynamics.

The model shows how agents requiring only 30% similar neighbors typically end up with 70-80% similar neighbors through spatial sorting. This amplification mechanism explains persistent residential segregation arising from preferences far milder than extreme prejudice.

These 20 interactive visualizations explore the model's mechanics, thresholds, dynamics, and extensions— revealing how local rules produce global patterns, how small preference changes near critical thresholds create dramatic outcomes, and how integration can remain stable below the tipping point.
