## Usage Notes

1. **Accessing Demos:** Navigate to `[gallery-path]/[demo-file].html`
2. **Gallery Indexes:** Most galleries have an `index.html` listing all demos
3. **Self-Contained:** All demos are standalone HTML files with inline CSS/JS
4. **CDN Dependencies:** External libraries loaded from CDNs (no local dependencies)
5. **Browser Requirements:** Modern browsers with Canvas/WebGL support
