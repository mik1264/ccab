Axelrod's Culture Model

Each agent has multiple cultural features (e.g., language, religion, values). When neighbors interact:

• Interaction probability = similarity
• One random differing feature is copied
• Result: "Cultural regions" emerge

Colors represent cultural similarity (computed from all features combined)
